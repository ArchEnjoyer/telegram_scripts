# telegram_scripts
Скрипты для чтения прошлых сообщений, отправленных в бот telegram, а также для спама реакциями от имени пользователя

частично навайбкожено :)
spam_reactions - ставит реакции (пользователь
view_last_messages - читает сообщения (пользователь)
parse_chats - показывает chat id всех диалогов (пользователь)
python-telegram-log-reading.py - читает сообщения в реальном времени (тг бот)
python-telegram-log-reading_with_logging.py - перенаправляет все сообщения боту из чата (тг бот)
